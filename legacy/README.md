# Legacy Prototype Files

This directory contains pre-EMNLP prototype code retained only for historical
inspection. It is not part of the EMNLP 2026 demo pipeline.

The current fixed-judge baseline is implemented by
`src.decision.baselines.fixed_judge_decision` and released through
`data/results/fixed_judge_results.csv`.

Obsolete QA-centered recording scripts, planning notes, reports and figures were
removed from the release package to keep the EMNLP 2026 materials focused on
the ESL literary-feedback storyline.
